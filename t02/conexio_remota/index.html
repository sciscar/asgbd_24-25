    Guia de Connexió

    1. Configuració de MongoDB

    1.1 Configurar l'autenticació

    Per defecte, MongoDB no està configurat per utilitzar autenticació. A continuació s'explica com habilitar-la.

    1. Crea un usuari administrador accedint a MongoDB des del terminal amb mongosh:

      Bash
      1
2
3
4
5
6
      use admin
db.createUser({
  user: "root",
  pwd: "Root_pass1",
  roles: [ { role: "root", db: "admin" } ]
})

    2. Obri l'arxiu de configuració mongod.conf. En Ubuntu, es troba generalment en:

      Bash
      1
      /etc/mongod.conf

    3. Edita la secció de security per habilitar l'autenticació:

      YAML
      1
2
      security:
  authorization: "enabled"

    4. Guarda els canvis i reinicia el servei MongoDB:

      Bash
      1
      sudo systemctl restart mongod

    5. Ix del shell de MongoDB i torna a connectar-te utilitzant el nou usuari administrador:

      Bash
      1
      mongosh -u "admin" -p "contrasenyaSegura" --authenticationDatabase "admin"

    2. Habilitar l'accés remot

    Per defecte, MongoDB només accepta connexions des de localhost. A continuació s'explica com habilitar l'accés remot:

    1. Obri l'arxiu de configuració mongod.conf:

      Bash
      1
      sudo nano /etc/mongod.conf

    2. Busca la secció net i modifica el paràmetre bindIp per permetre connexions des de qualsevol IP (això és perillós en entorns de producció, només recomanat en desenvolupament):

      YAML
      1
2
3
      net:
  port: 27017
  bindIp: 0.0.0.0

      També pots especificar una IP concreta en lloc de 0.0.0.0 per millorar la seguretat.

    3. Guarda els canvis i reinicia el servei MongoDB:

      Bash
      1
      sudo systemctl restart mongod

    4. Assegura't que el port 27017 està obert en el firewall del teu sistema:

      Bash
      1
      sudo ufw allow 27017

    3. Connexió remota amb mongosh

    Una vegada l'accés remot està habilitat, pots connectar-te des d'una altra màquina utilitzant mongosh.

    3.1 Instal·lació de mongosh

    Si no tens mongosh instal·lat, pots instal·lar-lo amb npm:

    Bash
    1
    npm install -g mongosh

    3.2 Connexió remota

    Per a connectar-te de manera remota, utilitza la següent comanda:

    Bash
    1
    mongosh "mongodb://admin:contrasenyaSegura@<IP_del_servidor>:27017/admin"

    Assegura't de canviar <IP_del_servidor> per la IP del servidor remot, i que el port 27017 està obert.

    4. Comandes bàsiques de MongoDB

    4.1 Crear una base de dades

    Bash
    1
    use nom_base_de_dades

    4.2 Crear una col·lecció

    Bash
    1
    db.createCollection("nom_col·lecció")

    4.3 Inserir un document

    Bash
    1
    db.nom_col·lecció.insertOne({ nom: "Joan", edat: 30 })

    4.4 Consultar documents

    Bash
    1
    db.nom_col·lecció.find({ nom: "Joan" })

    4.5 Actualitzar un document

    Bash
    1
    db.nom_col·lecció.updateOne({ nom: "Joan" }, { $set: { edat: 31 } })

    4.6 Eliminar un document

    Bash
    1
    db.nom_col·lecció.deleteOne({ nom: "Joan" })

    5. Seguretat i bones pràctiques

    • No exposes MongoDB directament a Internet sense configurar l'autenticació i un firewall adequat.
    • Canvia la configuració de bindIp per permetre només les IPs necessàries.
    • Utilitza contrasenyes fortes per als usuaris.
    • Considera utilitzar TLS/SSL per a xifrar les connexions entre clients i servidors MongoDB.
