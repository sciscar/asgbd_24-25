    Pràctica 2: Instal·lació i Configuració de MongoDB

    Objectiu de la pràctica

    Aquesta pràctica té com a objectiu avaluar els següents Resultats d'Aprenentatge (RA) i Criteris d'Avaluació (CA):

    RA1: Implanta sistemes gestors de bases de dades analitzant les seves característiques i ajustant-se als requeriments del sistema.

    CA: - a) Reconeixement de la utilitat i funció dels elements d'un SGBD. - b) Anàlisi de les característiques dels principals SGBD. - c) Selecció del SGBD adequat. - d) Identificació del programari necessari per a la instal·lació. - e) Verificació dels requisits hardware. - f) Instal·lació del SGBD. - g) Documentació del procés d'instal·lació. - h) Interpretació dels missatges d'error i fitxers de registre. - i) Resolució d'incidències durant la instal·lació. - j) Verificació del funcionament del SGBD.

    RA2: Configura el sistema gestor de bases de dades interpretant les especificacions tècniques i els requisits d'explotació.

    CA: - a) Descripció de les condicions d'inici i parada del SGBD. - c) Assegurament dels comptes d'administració. - d) Configuració de les eines i programari client. - e) Configuració de la connectivitat en xarxa. - h) Documentació del procés de configuració.

    RA3 (parcial): Implanta mètodes de control d'accés utilitzant assistents, eines gràfiques i comandaments del llenguatge del SGBD.

    CA: - c) Definició i eliminació de comptes d'usuari. - h) Garantia del compliment dels requisits de seguretat.

    Descripció de la pràctica

    L'objectiu és realitzar la instal·lació i configuració de MongoDB en un sistema Linux, seguint els passos detallats a continuació, i documentant tot el procés en format Markdown.

    Tasques a realitzar

    Anàlisi i selecció del SGBD:

    1. Investiga les característiques principals de MongoDB com a SGBD NoSQL.
    2. Justifica la seva elecció per a determinats tipus d'aplicacions o necessitats.
    3. Compara breument MongoDB amb altres SGBD NoSQL (com Cassandra, Redis) i SGBD relacionals (com MySQL).

    Preparació del sistema:

    1. Verifica els requisits hardware i software necessaris per a la instal·lació de MongoDB.
    2. Identifica i instal·la les dependències necessàries.
    3. Assegura't que el sistema està actualitzat (sudo apt-get update i sudo apt-get upgrade).

    Instal·lació de MongoDB:

    Opció A: Instal·lació utilitzant els binaris oficials. 1. Descarrega el paquet binari de MongoDB des del lloc web oficial. 2. Descomprimeix el paquet i mou el directori a la ubicació adequada (/usr/local/mongodb).

    Opció B: Instal·lació utilitzant el gestor de paquets APT. 1. Afegeix el repositori oficial de MongoDB al sistema. 2. Instal·la MongoDB utilitzant sudo apt-get install mongodb.

    Documenta els passos seguits i justifica l'elecció de la metodologia d'instal·lació.

    Configuració inicial de MongoDB:

    1. Crea els directoris necessaris per al funcionament de MongoDB (per exemple, /data/db).
    2. Configura el fitxer de configuració (/etc/mongod.conf) si és necessari.
    3. Assegura que MongoDB s'inicia automàticament com a servei del sistema.

    Inici i parada del servei MongoDB:

    1. Inicia el servei de MongoDB i verifica el seu estat.
    2. Detén i reinicia el servei, documentant els comandos utilitzats.
    3. Descriu les condicions i mètodes per a l'inici i parada del servei.

    Connexió a MongoDB i creació d'un usuari administrador:

    1. Conecta't al shell de MongoDB utilitzant el comandament mongo.
    2. Crea un usuari administrador a la base de dades admin amb una contrasenya segura.
    3. Documenta els comandos utilitzats i assegura que l'usuari té els permisos adequats.

    Asegurar MongoDB:

    1. Habilita l'autenticació a MongoDB per requerir usuaris i contrasenyes en les connexions.
    2. Configura el fitxer de configuració per aplicar aquesta seguretat.
    3. Reinicia el servei i verifica que l'autenticació està funcionant correctament.

    Proves de funcionament:

    1. Crea una nova base de dades de prova (testdb) i una col·lecció (users).
    2. Inserta documents a la col·lecció i realitza consultes per verificar que tot funciona correctament.
    3. Documenta les operacions realitzades i els resultats obtinguts.

    Instal·lació i ús de MongoDB amb Docker (opcional per a estudiants avançats):

    1. Descarrega la imatge oficial de MongoDB des de Docker Hub.
    2. Executa un contenidor de MongoDB utilitzant Docker, exposant el port adequat.
    3. Configura la persistència de dades mitjançant volums de Docker.
    4. Conecta't al MongoDB del contenidor i realitza operacions bàsiques.

    Documentació del procés:

    Elabora un document en format Markdown que inclogui tots els passos realitzats, les comandes utilitzades, captures de pantalla o còpies del terminal i explicacions clares. Inclou una secció de resolució d'incidències on expliquis els problemes trobats i com els has solucionat. Finalitza amb unes conclusions personals sobre l'experiència i els coneixements adquirits.

    Materials necessaris:

    • Un ordinador amb sistema operatiu Linux (pot ser una màquina virtual).
    • Connexió a Internet per a descarregar els paquets i documentació necessària.
    • Accés a una compte d'usuari amb permisos d'administrador (root o sudo).
    • Documentació oficial de MongoDB i recursos del Tema 2.
